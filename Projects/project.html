Hotel Room Booking System
Project Description:

The project aims to create a simple and user-friendly hotel booking webpage where users can browse and view details of various hotels in a specific location.

Features :
Hotel Listings :

Displays a 6 list of available hotels with images, names, locations, and a button to view more details.

User Interface :

A clean and organized layout with a modern design to enhance user experience.

Hotel Information :

Each hotel card provides a name, address, and an image, giving users a quick overview.

"View Details " Button :

Allows users to navigate to a detailed page with more information about the selected hotel.

Responsive Design :

Ensures the webpage is accessible on different devices, including desktops, tablets, and mobile phones.

Technologies Used :

HTML -> structuring the webpage.

CSS -> styling and layout design.

JavaScript > interactivity and navigation.

Page Features :
Hotel Image & Name :

The top section displays an image of the hotel along with its name, making it visually appealing for users.

Booking Form :
Check-in & Check-out Dates :

Users can select their preferred stay dates using a date picker.

Personal Information :

Fields to enter the guest's name and mobile number for communication.

Room Type Selection :

A dropdown menu allows users to select their preferred room type.

Number of Guests :

A field to specify how many people will be staying.

Price Field :

Displays the price based on selected preferences.

"Book Now " Button :

A call-to-action button that allows users to submit their booking request.
